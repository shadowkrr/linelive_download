# linelive_download

コマンド一発でLINELIVEのストリーミングデータを結合して１つの動画ファイルmp4に変換します

Combines LINELIVE streaming data into a single video file mp4 with a single command.


ffmpegが必要

require ffmpeg

[install document](http://shadowkrr.hateblo.jp/entry/2016/11/11/105512)
